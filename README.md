## Hur du använder detta repo

### Daglig träning
1. Kör igenom glosor i `vocabulary/`
2. Gör övningar i `exercises/`
3. Uppdatera din progress i `PROGRESS.md`

### Efter lektioner
1. Lägg anteckningar i rätt kursmapp
2. Lägg till nya glosor i `vocabulary/`
3. Notera läxor i `exercises/`

### AI-assistenten (Claude)
Denna assistant känner till din nivå och kan hjälpa dig med:
- Förklara grammatik
- Öva konversation
- Skapa övningar baserat på ditt material
- Testa dig på glosor
- Hjälpa med läxor
